## How to Print an Order

### Step 1: Wait for Order
- Order appears in kitchen dashboard automatically
- Red badge = pending (not yet started)

### Step 2: Click Print Button
- Click the **🖨️ Print** button on the order
- Small print dialog will appear (~1 second)

### Step 3: Select Printer
- Choose your kitchen printer from the list
- Usually labeled: "Kitchen Printer" or "Receipt Printer"

### Step 4: Print
- Click **Print** button in the dialog
- Order receipt prints to kitchen printer

### Step 5: Physical Ticket
- Take the printed receipt
- Read: Order number, table, and all items
- Start cooking!

## Troubleshooting

### Print Dialog Doesn't Appear
**Problem:** Clicked print, but nothing happened

**Solution:**
1. Check if your IT blocked popups
2. Ask admin to check browser settings
3. Try again in 5 seconds (sometimes slow)

### Print Preview Shows Blank
**Problem:** Dialog opens but receipt is empty

**Solution:**
1. Click "Cancel"
2. Refresh kitchen dashboard (F5)
3. Click print again on the order

### Can't Select Kitchen Printer
**Problem:** Printer not in the list

**Solution:**
1. Check that kitchen printer is powered on
2. Check it's connected to restaurant WiFi
3. Ask admin to reset printer connection
4. Restart the order print

### Order Won't Print
**Problem:** Print seems to work but nothing comes out

**Solution:**
1. Check printer has paper ✓
2. Check printer is powered on ✓
3. Check printer is online (green light) ✓
4. Try printing again
5. Ask admin for help

## Best Practices

✅ **DO:**
- Print immediately when order arrives
- Keep receipts organized by table number
- Stack oldest orders on top
- Match printed recipe to screen order status
- Discard receipt when "Serve" is marked

❌ **DON'T:**
- Wait to print (print right away!)
- Mix up table numbers
- Ignore printed orders
- Forget to mark as "Preparing" after printing
- Print multiple copies of same order

## Example Workflow

```
10:30 AM  Order arrives in kitchen (RED)
          ├─ Click 🖨️ Print
          ├─ Select Printer
          └─ Click Print → ✓
```

## Need Help?

**Ask Admin About:**

- Language settings
