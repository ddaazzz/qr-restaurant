QR Scan & Order

Customers scan a QR code at their table and instantly access your digital menu. No app installation required, no friction—just frictionless ordering from any smartphone.

  • ✓ Works on any smartphone - iOS, Android, regardless of browser
  • ✓ One-tap access - QR code instantly opens menu, no navigation needed
  • ✓ Table identification - Each QR is unique, system knows exactly which table ordered
  • ✓ Customizable QR codes - Generate new QR codes anytime with one click

Smart Menu Management

Create and manage your entire menu with beautiful images, detailed descriptions, and flexible pricing. Add variants like size, toppings, and extra options. Update items in real-time without restarting anything.

  • ✓ Photo uploads - High-quality images for every menu item
  • ✓ Variant management - Size, toppings, sauces, and more with dynamic pricing
  • ✓ Instant updates - Disable sold-out items immediately, no app refresh needed
  • ✓ Category organization - Group items logically for easy browsing

Table Management

Organize your restaurant layout, set table capacities, and track which tables are occupied. Monitor active dining sessions and generate unique QR codes for each table automatically.

  • ✓ Unlimited tables - Scale from 1 to 100+ tables
  • ✓ Table categories - Indoor, patio, bar—organize by location
  • ✓ Real-time occupancy - See which tables are active and how long they've been dining
  • ✓ Auto-generated QR codes - One click to create QR codes for all tables

Staff & Role Management

Create staff accounts with role-based permissions. Admin, staff, and kitchen roles with PIN-based authentication for kitchen staff. Track activity and monitor performance.

  • ✓ Three role types - Admin, staff, kitchen with specific permissions
  • ✓ PIN authentication - 6-digit PIN for quick kitchen staff login
  • ✓ Activity logging - Track who did what and when
  • ✓ Performance analytics - See which staff members are most active

Kitchen Display System

Powerful kitchen dashboard optimized for displaying orders. Kitchen staff sees incoming orders in real-time, can update order status, and manage cooking workflow efficiently.

  • ✓ Real-time order queue - Orders appear instantly as they're placed
  • ✓ Full item details - View all customizations, variants, and special requests
  • ✓ Status management - Update orders as pending, preparing, ready, or completed
  • ✓ PIN login - Quick access without password hassle
